O Aprendizado

O Início

A primeira coisa que aprendi na Trybe eu nem fazia parte dos alunos matriculados. Foi no curso introdutório, oferecido antes de realizar o teste prático para se matricular na Trybe, onde tive meu primeiro contato com JavaScript.

Durante a Trybe

Eu nem sabia para que servia o terminal direito. Git e GitHub eu nem sabia que existia. Finalmente entendi como HTML e CSS funcionam, além de aprender como deixar as páginas web mais acessíveis a qualquer pessoa que tiver interesse em acessar.

Momentos WoooW

Durante o Bloco 3 onde a empolgação foi a mil e eu estava adiantado no conteúdo, porém sempre quando eu não tinha mais opções para o aprimoramento do exercício a aula do Rods sobre um conteúdo que eu ja havia lido abriu minha mente para novos horizontes.
